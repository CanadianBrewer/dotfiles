The 
	.gitconfig 
	.bashrc
	.bash_profile
files go in the %USERPROFILE% directory

The 
	git-prompt.sh 
file goes in %PROGRAMFILES%\git\etc\profiled

The 
	RiderSetting.zip 
file should be extracted to
	%USERPROFILE%\.Rider2018.3\config\settings.zip
